## About

The Task Manager project demonstrates front-end architecture, Firebase
integration, and user data management using modern React techniques.  
Authenticated users can efficiently manage daily tasks with a responsive,
dynamic UI.  
This project is ideal for showcasing practical React skills, Firebase
Authentication, and clean component-driven development.

---

## ‚ö° Key Features

- **User Authentication:** Secure login and registration using Firebase
  Authentication.
- **Task Management:** Create, read, update, and delete tasks in real time.
- **Data Persistence:** User-specific tasks are stored in Firebase.
- **Dynamic UI:** Instant updates with React state management.
- **Responsive Design:** Tailwind CSS ensures consistent experience across
  devices.
- **Clean Code Structure:** Modular components, hooks, and reusable logic for
  scalability.

## Run Project Locally

Follow these steps to run the project on your local machine:

1. **Clone the repository:**

```bash
git clone https://github.com/ashikurahman1/task-manager-client.git
cd task-manager-client
npm install
Create a .env file in the root folder and add your Firebase config:
npm run dev
```
